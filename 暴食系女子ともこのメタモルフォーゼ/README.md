#命名規則

クラス、関数は大文字始まりのキャメル　

Update(),SceneManager,DoThisAtThisTime()的な


変数は小文字スネークで

variable, time_counter, very_long_snake_variable的な感じ


ファイル名はそのファイルで書かれてるクラス名を小文字始まりキャメルで

SceneManagerだったらsceneManager.hpp/.cpp

Titleだったらtitle.hpp/.cpp

SomeRandomClassだったらsomeRandomClass.hpp/.cppでオネシャス
